## [2.0.0] — Current Release

### ✅ Added

#### 🔢 Simple Mode
- Basic arithmetic: +, −, ×, ÷, %, +/−, AC, decimal point

#### 🔬 Scientific Mode
- Trigonometric: sin, cos, tan
- Logarithmic: log, ln
- Powers: x², x³, xʸ, √, 1/x
- Constants: π, e
- Factorial: n!
- Parentheses support

#### 💻 Programmatic Mode
- Base switching: DEC / HEX / OCT / BIN with live conversion display
- Bitwise operations: AND, OR, XOR, NOT
- Hex digit input (A–F)

#### 💰 Financial Mode
- EMI / Loan calculator (principal, rate, months)
- Compound Interest calculator (principal, rate, years, compounds/year)
- ROI calculator (initial vs final value)

#### 📏 Unit Conversion Mode
- 7 categories: Length, Weight, Temperature, Area, Volume, Speed, Time
- Bidirectional conversion with swap

#### 💱 Currency Mode
- Live exchange rates via public API
- Refresh button
- Swap currencies
- Rate hint display

#### 🔲 Matrix Mode
- 2×2 matrix A and B input
- Operations: A+B, A−B, A×B, det(A), A⁻¹, tr(A)

#### 📊 Statistics Mode
- Comma-separated data input
- Mean, Median, Mode, Standard Deviation, Variance, All Stats

#### 📅 Date & Time Mode
- Date difference calculator
- Add/subtract days from a date
- Timezone converter (10 timezones)

#### ❤️ Health Mode
- BMI calculator with category
- TDEE (calorie needs) with activity level
- Body Fat % via Navy Method (male/female)

#### 🎨 Color Mode
- Color picker with live preview
- HEX ↔ RGB ↔ HSL conversion
- Contrast checker with WCAG ratio
