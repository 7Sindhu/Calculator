# 🧮 Calculator

A feature-rich, multi-mode web calculator built with vanilla HTML, CSS, and JavaScript.

## Features

| Mode | Description |
|------|-------------|
| 🔢 Simple | Basic arithmetic operations |
| 🔬 Scientific | Trigonometry, logarithms, powers, factorials |
| 💻 Programmatic | Binary, Octal, Decimal, Hex with bitwise ops (AND, OR, XOR, NOT) |
| 💰 Financial | EMI, Compound Interest, ROI calculator |
| 📏 Unit | Length, Weight, Temperature, Area, Volume, Speed, Time conversions |
| 💱 Currency | Live exchange rates with offline fallback |
| 🔲 Matrix | 2×2 matrix operations (add, subtract, multiply, determinant, inverse, trace) |
| 📊 Statistics | Mean, Median, Mode, Std Dev, Variance |
| 📅 Date & Time | Date difference, date arithmetic, timezone converter |
| ❤️ Health | BMI, TDEE (calorie needs), Body Fat % (Navy Method) |
| 🎨 Color | HEX/RGB/HSL converter, color picker, WCAG contrast checker |

## Getting Started

No build tools or dependencies required.

1. Clone or download the repository
2. Open `index.html` in any modern browser

```bash
git clone https://github.com/7Sindhu/Calculator.git
cd Calculator
open index.html
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Input digit |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last character |
| `Escape` | Clear all |
| `( )` | Parentheses (Scientific mode) |
| `A–F` | Hex digits (Programmatic mode) |

## Project Structure

```
Calculator/
├── index.html      # UI layout and all calculator panels
├── calculator.js   # All logic and mode handling
└── styles.css      # Styling and theming
```
