**Calculator** is a free, open-source multi-mode web calculator built with pure HTML, CSS, and JavaScript.
11 modes in one app — from basic arithmetic to matrix operations, currency conversion, and health calculators.

## ✨ Modes

| Mode | Description |
|---|---|
| 🔢 Simple | Basic arithmetic: +, −, ×, ÷, %, +/− |
| 🔬 Scientific | sin, cos, tan, log, ln, √, x², x³, xʸ, 1/x, π, e, n!, parentheses |
| 💻 Programmatic | DEC/HEX/OCT/BIN with live conversion, AND/OR/XOR/NOT bitwise ops |
| 💰 Financial | EMI/Loan calculator, Compound Interest, ROI |
| 📏 Unit | Length, Weight, Temperature, Area, Volume, Speed, Time conversions |
| 💱 Currency | Live exchange rates with refresh, swap currencies |
| 🔲 Matrix | 2×2 matrix: A+B, A−B, A×B, det(A), A⁻¹, tr(A) |
| 📊 Statistics | Mean, Median, Mode, Std Dev, Variance from comma-separated data |
| 📅 Date & Time | Date difference, add/subtract days, timezone converter |
| ❤️ Health | BMI, TDEE (calorie needs), Body Fat % (Navy Method) |
| 🎨 Color | Color picker, HEX/RGB/HSL converter, contrast checker (WCAG) |

## 🚀 Getting Started

No installation required. Just open in a browser:

```bash
git clone https://github.com/7Sindhu/Calculator.git
cd Calculator
# Open index.html in any modern browser
```

Or simply double-click `index.html`.

## 🏗️ Project Architecture

```
Calculator/
├── index.html       # All 11 mode panels + navigation
├── calculator.js    # All calculation logic
└── styles.css       # Dark theme + responsive layout
```
